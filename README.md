# 🎵 CodeAlpha - Music Player

## 📌 Project Overview
This project is a responsive Music Player built using HTML, CSS, and JavaScript. It provides an interactive interface where users can play, pause, and navigate through songs with a clean and modern design.

## ✨ Features
- Play and Pause music
- Next and Previous track controls
- Progress bar with seek functionality
- Display of song title and artist
- Responsive design for desktop and mobile devices
- Smooth and user-friendly interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your preferred web browser.
4. Enjoy playing music using the built-in controls.
